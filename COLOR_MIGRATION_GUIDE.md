## Font Changes

### Body Font
- **Before**: System fonts (`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, ...`)
- **After**: `'HK Grotesk'` with system fonts as fallback

## Summary

The extension now uses a consistent Mixpanel brand identity with:
- **Primary Brand Color**: #4F45E0 (Mixpanel Purple) for all interactive elements and emphasis
- **Background/Menu Color**: #F6F6F6 (Light Gray) for input fields and navigation
- **Text Color**: #505055 (Dark Gray) for all readable text
- **Typography**: HK Grotesk for a sophisticated, modern, and highly legible aesthetic

All gradients have been replaced with solid colors for a cleaner, more professional look that aligns with Mixpanel's current brand guidelines.
